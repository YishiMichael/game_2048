# game_2048

*The web-based 2048 game*

Simply run `index.html`. Play for fun!

## Features

> Mlutiple dimentions  
> Scalable shape  
> Variable to set the number of tiles to merge

## Requirements

This repository depends on `jquery` and `Font Awesome`.
